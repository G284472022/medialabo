第2回 CSS による Web ページの装飾

今回はCSSを作成し、ホームページを見出し、画像、文体などで装飾をした。先週までの知識では色味のない最低限の情報しかないホームページしか作ることができなかったが、今回学んだCSSを作成するということでいつもインターネットで調べ物をした時などに見るようなホームページに少し近いようなものを作ることが出来るようになった。
