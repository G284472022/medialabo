「Webコンテンツの作成」の成果

本テーマの学習目的

本テーマの学習目的は、HTMLでのp要素やh1,h2のようなタグを使う記述の基本、CSSを使ってWebページの見た目の設定、JavaScriptによるWebページの処理を学び、その知識を組み合わせて簡単なWebページを作成できるようにすることである。

  • project.html

    • 数当てゲーム

    この数当てゲームは1~10の範囲での乱数を一つ出し、それを「答え」とする。そしてユーザは数字を最大3回入力して答えして「答え」を予想する。ユーザの入力が外れた場合、入力より「答え」が大きいか小さいかを表示する。最大3回の予想の中で「答え」を当てることができればユーザの勝利で当てることができなければユーザの負けというルールのゲームである。 第4回の改良では、数当てゲームの結果をコンソールではなく、ページに表示するように修正した。予想のメッセージをconsole.log()ではなくHTMLに出力し、hantei() の呼び出し回数を4回から1回に減らした。 第5回の改良では、テキストボックスに整数を指定して回答ボタンを押すと、正解判定の結果を表示し、正解すると正解というメッセージを表示し、4回以上回答するとゲームが終わっているというメッセージを表示するようにした。

    今回はWWWの技術やHTML文書の要素や構成やデバッグなどの基本的なことを学んだ。WWWやHTMLについての情報に触れることが初めてだったため、とても苦戦した。だが、諦めず理解し、今後の勉強、仕事などに活かしていきたいと考える。

    • 第2回 CSS による Web ページの装飾

    今回はCSSを作成し、ホームページを見出し、画像、文体などで装飾をした。先週までの知識では色味のない最低限の情報しかないホームページしか作ることができなかったが、今回学んだCSSを作成するということでいつもインターネットで調べ物をした時などに見るようなホームページに少し近いようなものを作ることが出来るようになった。

    • 第3回 JavaScript の基本

    今回はwedページを作る上で必要なJavaScriptについて初めて学んだ。オブジェクトの参照と代入や、配列の要素の一つずつに対して処理を行うfor-of構文、プログラムのデバッグなど、学んだことは非常に基本的なことだったが、javaと構文が似てるものがあるとはいえ、かなり難しく、慣れるためには、もっと時間と練習が必要だと感じた。

    • 第4回 DOM と JavaScript

    今回はDOMプログラミングについて学び始めた。今はまだJavaScriptにすらまったく慣れていない状況で、JavaScriptの基本をしっかり分かっていて初めてできるようになるであろうDOMプログラミングは今回の講義の中では到底理解することはできなかった。JavaScriptの基本と同じでかなりの練習量が必要だと感じた。

    • 第5回 イベント処理

    今回の講義ではイベント処理というものについて学んだ。まず、イベントとはそもそも何かということから始まり、文字列をボタンラベルにできるbutton要素、click要素、入力項目の説明するlabel要素、type,name,sizeなどのinput要素や、そこで入力された文字列を変数に代入できるvalueについて学んだ。
  • practice-event/event-intro.html
  • practice-event/event-list.html

    • 第6回 プロジェクト課題を完成させよう

    今回はAJAX通信について学んだ。AJAXとはページ全体を再読み込みすることなく、ページの一部を更新する処理で、XMLHttpRequest というオブジェクトを使った通信と、HTML, CSS, JavaScript, DOM, イベント処理の技術を組み合わせている。通信処理を実行するプログラムには、レスポンスが返ってくるまで待つ同期通信と、レスポンスの受け取りはイベント処理として記述する非同期通信の二つがある。
